\# Meeting Highlights System



\## Problem

In most organizations, meetings generate long transcripts and it becomes difficult to find important decisions and action items quickly. This leads to wasting time by re-discussing things.



\## Solution

This project helps in automatically extracting useful information from meeting recordings or transcripts. It generates transcript, summary, action items, and also provides a chatbot to ask questions from the transcript.



\## Features

\- Upload audio/video and get transcript

\- Generate summary

\- Extract action items

\- Chatbot to ask questions from transcript

\- Upload transcript separately

\- Extract decisions and action items

\- Download results as PDF and CSV



\## Tech Stack

\- Python (Flask)

\- React.js

\- Groq API

\- Whisper

\- FFmpeg



\## How to Run



\### Backend

cd backend  

python -m venv venv  

venv\\Scripts\\activate  

pip install -r requirements.txt  

python app.py  



\### Frontend

cd frontend  

npm install  

npm start
